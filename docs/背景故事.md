# 背景故事
这里属于自己扯了一个，后续有机会参照这个对原本进行重构

一个普通居民，拿起了木剑和藤甲，决心闯荡。与现实相同，第一次的他很快就被史莱姆给打败吞噬了，变成了骷髅。在xx年过后，苏醒了自己的意识。拾起了破旧的武器，遵循本能地战斗。骷髅阶段，其行动速度固定，防御力较高。只能够使用挥砍；当其击败巨型史莱姆后，将其包裹于自己的周身，借助史莱姆的润滑，可以实现跳跃和短冲刺，移动速度得到了提升；接着是遇到了精灵雇佣兵，其是感受到了自然被破坏的邪恶气息，参照死灵复生的设定。除却护甲外，史莱姆也可以作为一种特殊护甲，但其耗尽后会失去效果，变为原本的骷髅人。将精灵击败后，获取精灵之心，终于沉寂的魂火复苏了，恢复了神智。但是此时已然是xx年以后，为了找寻新的存在意义，决心去恶魔之地冒险。这个实际上受到了精灵之心的潜意识作用。在击败了最终boss后，看着自己的究极种族杂合体，决心故地重游，将其一一归还，故地的难度也会逐步减弱，因为此时的player也是逐步削弱的过程，最终回到那片森林，坐在洞穴岩壁边上，魂火陷入了沉寂。此时，惊醒，发现实际上还没出发，还是那个木剑的少年。此时可以结束游戏或者开启新周目，但此时是以人类之躯进行的，部分的技能树是有的，但没有解锁。这里的话，就是只有最终boss了。但是有彩蛋，回到森林，会发现有一个枯骨，触发记忆中的战斗。

## ai润色版本
<b>起源：求生之路</b><br>
不知为何而活。一个普通人，拿起木剑与藤甲，踏上了未知的征途。然而命运与现实如出一辙——初次冒险的他，很快倒在史莱姆的黏液下，化为游荡的骷髅。

<b>苏醒：遗忘与本能</b><br>
不知多少年后，枯骨中重新燃起微弱的魂火。他拾起锈蚀的武器，遵循战斗本能前行。骷髅之躯行动迟缓，却拥有较高的防御，仅能施展简单的挥砍。

<b>蜕变：借力前行</b><br>
当他击败巨型史莱姆后，将其黏液裹于周身，借助那层滑腻的介质，终于实现了跳跃与短距离冲刺，速度不再是他最大的短板。

<b>冲突：精灵的审判</b><br>
精灵雇佣兵感知到了自然之力的异常波动，认定这具不死的亡灵是邪恶的根源，率先发起了攻击。史莱姆黏液作为临时护甲会逐渐耗尽，一旦枯竭，他将再次变回脆弱的骷髅。

<b>复苏：记忆与使命</b><br>
击败精灵后，他夺取了“精灵之心”。沉寂的魂火彻底觉醒，记忆如潮水般涌来。然而世间已过百年，为了寻找新的存在意义，他毅然迈向恶魔之地——却不知这份渴望，正是精灵之心的潜意识驱使。

<b>终局：归还与轮回</b><br>
击败最终Boss后，望着自己这副融合了多种族特性的躯体，他决定踏上归途，将所有不属于自己的力量，一件件归还给故地。每归还一件，他便削弱一分；故地的挑战，也随之变得温柔。最终，他回到最初的森林，靠着洞穴岩壁坐下，魂火逐渐暗淡、沉寂。

<b>惊醒：庄周梦蝶</b><br>
猛然睁眼——他还是那个手持木剑的少年，一切尚未开始。此时，玩家可以选择结束，或是开启新周目。但这一次，以人类之躯再临，技能树虽在，却大多尘封未解。最终Boss仍在原处等待，而在那片森林里，多了一具枯骨——触碰它，便能唤起前世的战斗记忆，亦是一场无声的彩蛋。
